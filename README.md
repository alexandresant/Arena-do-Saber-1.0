# 🎮 Arena do Saber

Plataforma educacional gamificada com elementos de RPG, desenvolvida com o objetivo de aumentar o engajamento e a motivação de estudantes no processo de aprendizagem.

Este projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) em Engenharia de Software.

---

## 📌 Sobre o Projeto

O **Arena do Saber** é uma plataforma de estudos que incorpora mecânicas de jogos de RPG, como:

- Sistema de personagens
- Evolução de atributos
- Missões e atividades
- Sistema de pontuação e experiência
- Ranking de estudantes
- Sistema de batalhas entre personagens

A proposta é transformar o processo de aprendizagem em uma experiência mais envolvente, utilizando elementos lúdicos para estimular a participação ativa dos estudantes.

## 🧠 Objetivo do Projeto

Investigar o impacto da gamificação baseada em elementos de RPG no:

- Engajamento dos estudantes
- Motivação para estudo
- Participação em atividades educacionais
